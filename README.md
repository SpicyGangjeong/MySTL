# MySTL
CPP STL을 모작합니다.

목표
1. STL 내부 구조 파악하기
2. 보편참조( 퍼펙트 포워딩 ) 이해하기
3. 자료구조 및 알고리즘 연습하기
4. 이동, 복사 연산자 발생상황 익히기
5. 최적화 공부하기
6. 장단점, 개선사항 고민해보기
7. 구현별로 인터페이스 통일하기
8. 할 수 있으면 메모리 덤프까지

제한사항 ( 모래주머니 )
- 경고수준 w4 - 경고를 오류로 표시 on
- C++17, Release모드 only
- std 사용 최소화
- pch 사용x 템플릿에 의한 빌드 시간관리도 해보자
- 정적라이브러리로 배포
- STL에서 사용하는 모든 기능 구현x 보편적인 기능만

현재구현
벡터 ( std::forward, std::move만 사용. )
- forward와 move에 있는 remove_reference_t 더 공부하고 내가 만든걸로 바꿔야함.
- 일단 보편참조는 이해함.
우선순위 힙 ( STL::Vector를 드라이버로 사용 -> MySTL::Vector로 바꿔야함 )
- 벡터 끝나면 힙 재구현 필요
- CreateHeap 쓸 수 있을까? 고민 필요.
